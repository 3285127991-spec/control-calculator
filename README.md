## 控制计算器（Control Calculator）
自动控制原理辅助计算工具，支持二阶系统参数计算、根轨迹分析和伯德图分析。
## 功能
- 二阶系统参数计算
  - 自然频率 ωn
  - 阻尼比 ζ
  - 超调量 Mp
  - 峰值时间 tp
  - 调整时间 ts

- 根轨迹分析
  - 根轨迹绘制
  - 开环极点/零点显示
  - 闭环极点计算

- 伯德图分析
  - 幅频特性
  - 相频特性
  - 增益裕度
  - 相位裕度

- JavaScript
